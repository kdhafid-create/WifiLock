# Wifi Lock — تطبيق مسح وتثبيت نقاط اتصال Wi-Fi

## ماذا يفعل التطبيق
1. يمسح كل نقاط Wi-Fi القريبة ويعرض لكل واحدة: الاسم (SSID)، المعرّف الفريد (BSSID)، وقوة الإشارة.
2. عند الضغط على "تثبيت" بجانب أي نقطة، يطلب كلمة مرور الشبكة (إن وُجدت) ويحاول إجبار الاتصال بهذه النقطة تحديداً باستخدام BSSID الخاص بها، حتى لو كانت هناك نقاط أخرى بنفس الاسم وإشارة أقوى.
3. زر "إلغاء التثبيت" يعيد الجهاز للسلوك التلقائي العادي.

## كيفية التشغيل
1. ثبّت [Android Studio](https://developer.android.com/studio) (مجاني).
2. افتح Android Studio → File → Open → اختر مجلد `WifiLock` هذا بالكامل.
3. انتظر حتى ينتهي Gradle من التزامن (Sync) تلقائياً (يحتاج إنترنت أول مرة لتحميل المكتبات).
4. وصّل هاتفك عبر USB وفعّل "خيارات المطورين" + "USB Debugging"، أو استخدم محاكي (Emulator).
5. اضغط زر التشغيل (▶) الأخضر أعلى الشاشة.

## قيود يجب معرفتها (بدون Root)
- التثبيت يعمل فقط طالما التطبيق نشط (foreground) أو له صلاحية استمرار بالخلفية.
- النظام قد يعتبر الشبكة "محلية فقط" في بعض الأجهزة/الإصدارات؛ الكود يحاول ربط حركة بيانات الجهاز بالشبكة المطلوبة عبر `bindProcessToNetwork` لتفادي هذه المشكلة قدر الإمكان.
- عند إغلاق التطبيق بالكامل من الخلفية (Force close/Swipe away)، يُلغى التثبيت تلقائياً ويعود الهاتف للتنقل الطبيعي بين النقاط.
- هذا سلوك مقصود من أندرويد لأسباب أمنية؛ التثبيت الدائم الكامل (حتى بعد إغلاق التطبيق) يتطلب صلاحيات Root أو أن يكون التطبيق "System App".

## الأذونات المطلوبة
- الموقع (ACCESS_FINE_LOCATION): مطلوب من أندرويد لعرض نتائج مسح Wi-Fi (حتى لو لم يستخدمها التطبيق فعلياً لتحديد الموقع).
- NEARBY_WIFI_DEVICES: بديل الموقع على أندرويد 13 فما فوق.
